## Standard Combo 1

### Description 
The base combo, built around high DPS with minimal protection. It uses as many tools as possible during CC windows and can be initiated from multiple engages such as a grab, Shadow Stomp, or Shadow Slash. Core skills can enhance it further. Contains many gaps, so it is best suited for 1v1 fights under controlled conditions.

### Inputs w/ Move Names

Death’s Descent (**E**) → Flashing Light (**Quick Slot**) → Murderous Intent (**Shift + LMB**) → Drastic Measure (**Shift + RMB**) → Flow: Execution (**Quick Slot**) → Sudden Decapitation (**S + F**) → Seamless (**Space**) → Serpent Ascension (**S + F**) → Vacuum Slash (**F**) → Ghost Step (**Shift + S**) → Corrupt Sword Dance (**S + LMB + RMB**) → Ghost Step (**Shift + S**) → Illusion of Restraint (**Quick Slot**)

### Inputs w/ Keybinds

**E → Quick Slot → Shift + LMB → Shift + RMB → Quick Slot → S + F → Space → S + F → F → Shift + S → S + LMB + RMB → Shift + S → Quick Slot**

## Standard Combo 3

### Description

A variation with Core Katana Shower for situational protection and a wider AoE on the second CC. Useful in group PvP for extra catches and area control.

Shadow Slash (**Q**) → Flashing Light (**Quick Slot**) → Murderous Intent (**Shift + LMB**) → Drastic Measure (**Shift + RMB**) → Mach Explosion (**RMB**) → Ghost Step (**Shift + A**) → Core Katana Shower (**Shift + F**) → Chaos Spree (**S + E**) → Illusion of Restraint (**Quick Slot**)

### Inputs w/ Keybinds

**Q → Quick Slot → Shift + LMB → Shift + RMB → RMB → Shift + A → Shift + F → S + E → Quick Slot**


## Standard Combo 4

### Description

Standard combo reworked with Core Drastic Measure in place of Flow: Execution. This increases protection while still maintaining reliable Knockdowns, making it ideal for small-scale fights and 1v1 scenarios.

### Inputs w/ Move Names

Shadow Slash (**Q**) → Flashing Light (**Quick Slot**) → Murderous Intent (**Shift + LMB**) → Core Drastic Measure (**Shift + RMB**) → Mach Explosion (**RMB**) → 50% BSR Chaos Spree (**S + E + RMB**) → Illusion of Restraint (**Quick Slot**)

### Inputs w/ Keybinds

**Q → Quick Slot → Shift + LMB → Shift + RMB → RMB → S + E + RMB → Quick Slot**
